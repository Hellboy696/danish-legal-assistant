# 🚀 ЯК ЗАПУСТИТИ STREAMLIT UI

## Пояснення (дуже просто):

Streamlit = це як запустити веб-сайт на своєму комп'ютері!

**Не треба:**
- Складного серверу
- HTML/CSS писати
- JavaScript знати

**Просто:**
- Запускаєш одну команду
- Відкривається браузер
- Бачиш красивий інтерфейс!

---

## 📋 КРОК ЗА КРОКОМ

### Крок 1: Переконайся що все встановлено

```bash
pip install streamlit sentence-transformers lancedb
```

**Що це робить:**
- Встановлює Streamlit (для UI)
- Встановлює sentence-transformers (для AI)
- Встановлює LanceDB (для пошуку)

---

### Крок 2: Перейди в папку проекту

```bash
cd "D:\Навчання\.venv\навчання з Claude\danish_legal_mvp"
```

**Пояснення:**
- `cd` = change directory (зміни папку)
- Ідеш в папку де лежить streamlit_app.py

---

### Крок 3: Запусти Streamlit!

```bash
streamlit run streamlit_app.py
```

**Що відбувається:**
1. Streamlit запускається
2. Завантажує Danish закони (30 секунд)
3. Відкриває браузер автоматично!
4. Бачиш інтерфейс з полем для питання

## 💡 ПРИКЛАДИ ПИТАНЬ ЩОБ СПРОБУВАТИ

### Питання 1: Про роботу
```
Can I work in Denmark with Ukrainian passport?
```
**Очікуваний результат:**
- Immigration Act §42
- Про work permit
- 2-3 місяці processing

### Питання 2: Про податки
```
What is the income tax rate in Denmark?
```
**Очікуваний результат:**
- Tax Law §123
- 37% до 52%
- Залежить від доходу

### Питання 3: Про відпустку
```
How many vacation days do I get per year?
```
**Очікуваний результат:**
- Holiday Act §7
- 25 днів (5 тижнів)
- Paid vacation

## 🔧 ЯКЩ ОЩО НЕ ПРАЦЮЄ

### Помилка: "streamlit: command not found"
**Рішення:**
```bash
pip install streamlit
```

### Помилка: "ModuleNotFoundError: No module named 'sentence_transformers'"
**Рішення:**
```bash
pip install sentence-transformers
```

### Помилка: "File not found: danish_laws.json"
**Рішення:**
- Переконайся що ти в правильній папці:
```bash
cd "D:\Навчання\.venv\навчання з Claude\danish_legal_mvp"
```
- Перевір що файл існує:
```bash
dir data\danish_laws.json
```

### Браузер не відкривається автоматично
**Рішення:**
- Подивись в консолі, там буде адреса типу:
```
You can now view your Streamlit app in your browser.

Local URL: http://localhost:8501
Network URL: http://192.168.1.xxx:8501
```
- Скопіюй `http://localhost:8501` і відкрий в браузері

## 🎮 ЯК КОРИСТУВАТИСЯ UI

### Базове використання:
1. Напиши питання в полі
2. Натисни "🔍 Шукати"
3. Дивись результат!

### З категорією:
1. Напиши питання
2. Вибери категорію (Immigration/Tax/Labor)
3. Натисни "🔍 Шукати"
4. Побачиш тільки результати з цієї категорії

### Копіювання тексту:
1. Після пошуку бачиш результати
2. Натисни "📋 Копіювати текст #1"
3. Текст з'явиться в текстовому блоці
4. Можеш скопіювати Ctrl+C
